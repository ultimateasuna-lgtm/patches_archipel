# Archipel Patch

Ce repository contient le fichier `patch.json` pour l'Archipel d'Epsilon.

## Informations
- **Nom**: Archipel
- **Version**: MU-2
- **Fichier**: patch.json (81939 lignes)

## Instructions pour créer le repository GitHub

1. Installez Git si ce n'est pas déjà fait : https://git-scm.com/download/win

2. Ouvrez un terminal dans ce dossier et exécutez :
```bash
git init
git add .
git commit -m "Initial commit: Archipel patch.json"
```

3. Créez un nouveau repository sur GitHub :
   - Allez sur https://github.com/new
   - Nommez-le "patches_archipel" ou un nom de votre choix
   - Ne cochez PAS "Initialize with README"
   - Cliquez sur "Create repository"

4. Liez votre dossier local au repository GitHub (remplacez YOUR_USERNAME) :
```bash
git remote add origin https://github.com/YOUR_USERNAME/patches_archipel.git
git branch -M main
git push -u origin main
```

## Alternative sans Git installé

Vous pouvez aussi :
1. Créer un nouveau repository sur GitHub
2. Glisser-déposer le fichier `patch.json` directement depuis l'interface web de GitHub
